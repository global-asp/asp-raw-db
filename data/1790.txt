desc:	We can do many actions with our bodies

page_text:

P1	I can run with it.
P2	I can jump with it.
P3	I can dance with it.
P4	I can swim with it.
P5	I can skip with it.
P6	I can kick with it.
P7	I can escape with it.
P8	But I can never fly with it.
