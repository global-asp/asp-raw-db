P1	Cuidado,  olha para ali. Aquele boi está zangado contigo. Corre para aqui!
P2	Nós vamos jogar contigo. Primeiro salto eu. Vocês saltam depois de mim.
P3	Há muitos pássaros a voar no céu. Alguns pássaros são pequenos e outros são grandes. Quantos pássaros há no céu?
P4	Chama todos os alunos! Nenhum pode faltar! Este é um dia especial. Toda a nossa turma vai tomar chá no pátio da escola.
P5	Repara como aquelas crianças gostam de dançar. Cada uma saltita para cima e para baixo. Cada uma saltita para trás e para a frente.
P6	Aquela árvore tem flores bonitas. Não,  não colhas essas flores. Ninguém deve colher flores.
P7	O que é que o titio vai comer ao jantar? Gostaria de comer um frango maior ou um frango menor?
P8	Qual é a tua comida preferida? Tu queres comer alguma coisa salgadaou doce? Gostarias desta galinha ou daquele doce?
P9	Olha! Há dois leões de baixo daquela árvore.Há uma leoa e um leão. Ele está à frente. Ela está atrás.
P10	Temos aqui dois balões para dar. Este balão é para ele e o outro é para ela.
P11	Temos aqui dois balões para dar. Este balão é para ele e o outro é para ela.
P12	Temos aqui dois balões para dar. Este balão é para ele e o outro é para ela.
P13	Pai,  quando é que vai ler um livro para mim? Eu gosto de ouvir o pai a ler!
P14	Eles têm pão fresco,  acabado de fazer. O pão ainda está quente. Vamos comprar alguns pães,  antes que os levem daqui!
P15	Onde é que conseguiste essas papaias? Estas papaias são da nossa papaeira.
P16	Para quem é que estás a escrever? Estou a escrever para a minha mãe. O que é que estás escrever? Estou a escrever um poema para dizer que a amo!
P17	Esta praia é boa para nadar e pescar? Sim! Vocês podem nadar e pescar aqui. Estamos felizes por podermos nadar e pescar nesta praia!
P18	Este grande tractor vai remover as rochas. Estão a fazer ali uma estrada nova. Depois as pessoas vão passar pela estrada nova.
P19	Esta publicação tornou-se possível graças ao apoio do povo americano através da Agência dos Estados Unidos para o Desenvolvimento Internacional (USAID).
