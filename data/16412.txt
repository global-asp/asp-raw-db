page_text:

P1	Esta é a Agnes. É a minha professora.
P2	Só consegue andar com muletas.
P3	Este é o Metobo. É DJ na rádio.
P4	O Metobo é albino. Tem problemas de visão.
P5	Esta é a Moraa. É agricultora.
P6	A Moraa é surda. Para falar,  usa linguagem gestual.
P7	Este é o Osero. É o melhor jogador de futebol da minha escola.
P8	O Osero teve um acidente com as mãos,  mas consegue ganhar jogos com os pés!
